## 📂 Overview
This dataset provides detailed records of **Delhi Metro trips**, including passenger counts, fares, distances, and ticket types. It is intended for transport research, operational analysis, and building data-driven insights into metro usage patterns.

---

## 📑 File Information
- **Filename**: `delhi_metro_.xlsx`
- **Format**: Excel workbook (`.xlsx`)
---

## 📊 Data Dictionary
| Column Name       | Description                                                                 |
|-------------------|-----------------------------------------------------------------------------|
| `Date`            | Unique identifier for each trip record                                      |
| `Date`            | Calendar date of observation (YYYY-MM-DD)                                   |
| `From_Station`    | Origin station name                                                         |
| `To_Station`      | Destination station name                                                    |
| `Distance_km`     | Distance traveled in kilometers                                             |
| `Fare`            | Fare charged for the trip (INR)                                             |
| `Cost_per_passenger` | Average cost per passenger for the trip                                  |
| `Passengers`      | Number of passengers recorded per day                                       |
| `Ticket_Type`     | Type of ticket used (Smart card, single, etc.)                              |
| `Remarks`         | Additional notes or comments (e.g., weekend, festival)                      |
